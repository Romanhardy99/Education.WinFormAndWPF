# Дескриптор
hwnd -> handler window 

window - окно;
handler - дескриптор;
дескриптор - это уникальный идентификатор / ключ / токен. 
 1. проверяет права
 2. изолирует процессы
 3. считает ссылки
 4. защищает от подделки handle
 5. может перемещать / перестраивать структуры (PTR != handler)
 -> выдает ОС window;
ОС Window -> менеджер объектов -> таблицу объектов ->  структура объекта (параметры
(размер и позиция стиль
указатель на оконную процедуру
очередь сообщений
дочерние окна
привязка к процессу
контекст GDI
состояние(видимо, активно, скрыто…)) ->
1) выделила память под структуру объекта
2) инициализировала её поля
3) добавила объект во внутренние таблицы / списки
4) создала handle, связанный с этим объектом
5) связала объект с процессом / сессией)
